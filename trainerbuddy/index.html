Worky

Att träna socialt med andra i naturen är viktigt för hälsan och för dig som person. Vi för samman folk som vill träna för att förbättra deras prestanda och pusha varandra till dne nästa nivån, eller kanske bara vör att få motivationen att gå ut och träna.

Med val av vad du vill träna hittar Worky andra som vill träna på det och kan ge förslag på vart man kan träna. Detta gör att du inte behöver söka upp lämpliga ställen själv, det gör Worky åt dig.

Worky jobbar med olika idrotts affärer runt sverige för att du ska kunna få rabbater på tränings verktyg för att maximera din träning.

Worky kan hjälpa dig med ett tränings programm som passar just dig. Ovansätt vilken nivå du ligger på så kan Worky skapa ett program som passar.
